# SimpleTaskList
Projeto em Java para gerenciamento de tarefas (To‑Do List) utilizando Spring Boot, JPA/Hibernate e banco de dados. O objetivo é aplicar conceitos de desenvolvimento web, persistência de dados e orientação a objetos.

## Descrição
O SimpleTaskList permite criar, listar, concluir e remover tarefas de forma estruturada, demonstrando boas práticas de desenvolvimento com Spring Boot, persistência com JPA/Hibernate e uso de enums e DTOs.

## Funcionalidades
- Criar novas tarefas
- Listar todas as tarefas
- Marcar tarefas como concluídas
- Remover tarefas
- Persistência de dados via banco relacional
- Estrutura RESTful para futuras integrações

## Tecnologias
- Java 17+
- Spring Boot
- Spring Data JPA / Hibernate
- Banco de dados H2 (testes)
- Maven (gerenciamento de dependências)
